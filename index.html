Trade Rep Dashboard

Это Next.js-приложение, а не один статический HTML-файл. HTML генерируется локальным dev-сервером.

Как открыть

  1. Установите зависимости: npm install
  2. Запустите сайт: npm run dev
  3. Откройте: http://localhost:3000
npm install
npm run dev

Где код страниц

  • app/page.tsx — главная Dashboard Overview
  • app/sales/page.tsx — Sales
  • app/group-plan/page.tsx — Group Plan
  • app/receivables/page.tsx — Receivables
  • app/tires/page.tsx — Tire Analytics
  • app/settings/page.tsx — Settings
